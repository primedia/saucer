# Saucer

Gem to export data from SauceLabs into Various Formats (currently XLSX)

## Installation

Add this line to your application's Gemfile:

    gem 'saucer'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install saucer

## Usage

    `cp .envrc.example .envrc`
    `source .envrc && ./bin/sauce`
